# 🧬 Pareamento de DNA

Este projeto implementa um algoritmo simples para realizar o **pareamento de uma sequência de DNA**, seguindo as regras da biologia molecular.

## 📌 Descrição

O programa recebe uma sequência de DNA e gera sua sequência complementar com base nas seguintes regras:

- Adenina (A) ↔ Timina (T)
- Citosina (C) ↔ Guanina (G)

Esse tipo de pareamento representa a estrutura básica da dupla hélice do DNA.

## 🚀 Funcionalidade

- Receber uma sequência de DNA
- Gerar a sequência complementar correspondente
- Exibir o resultado do pareamento

## 🛠️ Tecnologias

- Linguagem: C
- Execução via terminal
